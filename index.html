What Am I Doing?

This page and the projects I am working on the side are to both help me knock some of the rust off, learn new tools and techniques, and stay abreast of the details. Some examples of what I am currently exploring are:

  • Using modern tools like Git/GitHub, Visual Studio 2017, Anaconda, Eclipse, etc.
  • Using new languages and later versions of languages I currently know, including Python, R, .Net Core, Java etc.
  • Implementing modern concepts like Responsive Design, Microservice Architecture, Deep Learning, and DevOps Pipelines.

My goal is to gain a deeper understanding of the implementation tasks and technologies that I commonly espouse at the strategic and enterprise levels so that I can better work the full range and stack of issues for my clients. I also hope to be able to work at any level in any project that I am interested in.
